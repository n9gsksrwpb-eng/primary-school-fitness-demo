# 小学 1–6 年级匿名体测查询 Demo 产品规格

## 1. 项目名称

小学生匿名体测查询与健康教育助手

## 2. 项目定位

本项目是一个面向中国小学 1–6 年级学生、家长和体育教师的匿名体测查询 Demo。

用户输入匿名体测数据后，系统即时计算：

- BMI
- BMI 状态
- 单项得分
- 标准分
- 附加分
- 总分
- 体测等级
- 主要短板
- 基础运动建议

本项目不建立学生档案，不保存历史记录，不收集个人身份信息。

## 3. 非目标

本项目不做以下事情：

- 不做学生健康档案系统
- 不做学校体测管理后台
- 不做班级统计
- 不做历史趋势追踪
- 不做医学诊断
- 不做治疗建议
- 不做减肥方案
- 不做处方化运动干预
- 不收集姓名、学号、学校、班级、手机号等身份信息

## 4. 目标用户

### 4.1 学生

希望了解自己的体测项目表现，以及可以如何安全提升。

### 4.2 家长

希望理解孩子的体测结果，知道如何在日常生活中配合体育锻炼。

### 4.3 体育教师

希望快速生成单个学生的匿名体测解读样例，用于教学说明或健康教育。

## 5. 使用场景

### 场景一：完整体测查询

用户输入完整项目成绩，系统输出完整报告。

示例：

```text
三年级男生，身高136cm，体重32kg，肺活量1800ml，50米10.8秒，坐位体前屈7.2cm，跳绳95个，仰卧起坐32个。

系统输出：

BMI：xx
BMI 状态：正常/低体重/超重/肥胖
标准分：xx
附加分：xx
总分：xx
等级：优秀/良好/及格/不及格
单项得分表
短板分析
4周基础运动建议
```

### 场景二：缺项提醒

用户输入不完整数据时，系统说明当前年级需要哪些项目。

示例：

```text
四年级女生，身高140，体重34，跳绳100个。

系统提示：

四年级完整体测报告还需要：
- 肺活量
- 50米跑
- 坐位体前屈
- 一分钟仰卧起坐

你可以补充这些数据；如果暂时没有，我也可以先分析已输入项目，但不能计算完整总分。
```

### 场景三：非完整分析

用户不想补全项目时，系统仅分析已输入项目，并明确说明不能计算完整总分。

### 场景四：健康教育问答

用户询问某个短板项目如何提升。

系统根据通用体育教学原则给出安全、基础、非医疗化建议。

| 字段 | 类型 | 说明 |

## 7. 禁止输入字段

页面、接口、表单 schema 中不得出现以下字段：

| 禁止字段 | 原因 |
| --- | --- |
| name | 姓名属于个人身份信息 |
| studentId | 学号可识别具体学生 |
| school | 学校信息可能增强识别性 |
| className | 班级信息可能增强识别性 |
| phone | 手机号可直接识别个人 |
| idCard | 身份证号高敏感 |
| address | 地址高敏感 |
| birthday | 精确生日可能增强识别性 |
| photo | 照片涉及肖像或生物识别风险 |
| video | 视频涉及肖像或生物识别风险 |

## 9. 得分规则

### 9.1 标准分

标准分由各单项分数按权重加权得到，满分 100。

### 9.2 附加分

小学阶段一分钟跳绳可产生附加分。

附加分最高 20。

### 9.3 总分

总分 = 标准分 + 附加分

总分最高 120。

### 9.4 等级

| 总分 | 等级 |
| --- | --- |
| >= 90.0 | 优秀 |
| 80.0–89.9 | 良好 |
| 60.0–79.9 | 及格 |
| <= 59.9 | 不及格 |

## 10. 输出报告

完整报告包含以下部分：

### 10.1 匿名体测总览

展示：

- 年级
- 性别
- BMI
- BMI 状态
- 标准分
- 附加分
- 总分
- 等级

不得展示姓名、学校、班级等身份信息。

### 10.2 单项得分表

字段：

| 字段 | 说明 |
| --- | --- |
| 项目 | BMI、肺活量等 |
| 原始成绩 | 用户输入值 |
| 单项分 | 标准表匹配分 |
| 权重 | 对应年级权重 |
| 权重得分 | 单项分 × 权重 |
| 解读 | 简短解释 |

### 10.3 主要短板

系统自动找出 1–3 个需要优先关注的项目。

短板判断建议：

- 单项分 < 80：优先关注
- BMI 非正常：优先关注
- 多个低分项目时，按单项分从低到高排序

### 10.4 4周基础运动建议

建议按短板类型生成：

- BMI 偏低
- BMI 超重/肥胖
- 肺活量不足
- 速度能力不足
- 柔韧性不足
- 跳绳能力不足
- 腰腹力量不足
- 耐力不足

每周建议包含：

- 训练目标
- 推荐动作
- 每次时长
- 每周频次
- 注意事项

### 10.5 安全提示

必须包含：

本报告仅用于体育教学和健康教育参考，不构成医学诊断、治疗或处方。如运动中出现胸痛、头晕、呼吸困难、异常疲劳等情况，应停止运动并咨询医生、校医或专业体育教师。
